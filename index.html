コースのご紹介

1レッスンが3分でまとめられており、通勤や通学、家事のスキマ時間で学習できます。 動画学習のため、無理なくスキルを身につけられるようになっています。

デザインコース

デザインの基礎やAdobe Photoshop・Illustratorの使い方を学習します。 架空のプロジェクトを通して、Webデザイナーとしてのポートフォリオを作成できます。

マーケティングコース

デジタルマーケティングの基礎や広告戦略、データ分析を学習します。 実際のキャンペーン企画に参加して、マーケッターの実務を経験できます。

プログラミングコース

Webサイト制作に必要なHTML・CSSを学習します。架空のサイト制作を通して、 必要なアニメーション設定をJavaScriptで開発できるようになります。

コースの対象は?

このコースは、Web デザイン、開発、デジタル マーケティングのスキルと 知識を高めたいと考えている個人向けに設計されています。熟練した Web デザイナー、 新進の Web 開発者、またはスキル セットの拡張を目指すマーケティング プロフェッショナルなど、 どなたでも受講できます。

  • Web デザイナー
  • Web 開発者
  • マーケティング
  • フリーランス希望者
  • 副業エンジニア

期待される学習成果

コース修了後にどんなメリットが期待できるかご紹介します。 実際のコース卒業生たちのアンケートを見てみましょう。

デザイン基礎 - Adobe Photoshop・Illustrator・XD、Figmaのデザインソフトの基本操作が身につく。

UI/UXデザイン - ユーザー希望に合わせた、ワイヤーフレームとデザインカンプの提案力が身につく。

広告戦略 - デジタル広告の知識があり、プロジェクトの課題や売上目標に沿った広告戦略が立てられる。

データ分析 - Google Analyticsが使用でき、データの可視化やKPIの設定、モニタリング・評価ができる。

ウェブ開発 - HTMLとCSS、JavaScriptで、レスポンシブ対応のWebサイトをコーディングできる。

ポートフォリオの作成 - 架空のサイト開発の発注書を見て、デザイン、開発、公開まで行うことができる。
